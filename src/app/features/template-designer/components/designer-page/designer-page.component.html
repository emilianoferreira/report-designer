Zureo Report Designer

Bienvenido al diseñador de plantillas de comprobantes. Esta herramienta permite crear y personalizar el formato de impresión de facturas, tickets y otros documentos comerciales.

Estructura del diseñador

La interfaz se divide en tres paneles:

  • Panel izquierdo (Herramientas): contiene los elementos que se pueden arrastrar al canvas.
  • Panel central (Canvas): es la superficie de diseño donde se posicionan los elementos sobre la hoja.
  • Panel derecho (Propiedades): muestra y permite editar las propiedades del elemento seleccionado.

Los paneles laterales se pueden colapsar usando las flechas en sus bordes para ganar espacio de trabajo.

Secciones del comprobante

Cada plantilla tiene tres secciones:

  • Encabezado: datos de la empresa, tipo de comprobante, datos del cliente.
  • Detalle: líneas de productos/servicios (se repite por cada ítem).
  • Pie: totales, impuestos, observaciones y datos fiscales.

Elementos

Los elementos disponibles se agrupan en tres categorías:

Básicos

  • Texto: texto estático (etiquetas, títulos). Se escribe el contenido directamente.

Datos

  • Campo: campo vinculado a un dato del comprobante (nombre del cliente, fecha, etc.).
  • Fórmula: campo calculado con expresiones matemáticas.
  • QR / Código de barras: elementos gráficos generados a partir de datos.

Visuales

  • Imagen: logo u otra imagen estática o vinculada a datos.
  • Línea: separador horizontal, vertical o diagonal.
  • Rectángulo: forma rectangular con borde y/o relleno.

Agregar elementos

Arrastre un elemento desde el panel de herramientas y suéltelo sobre la sección deseada del canvas. El elemento aparecerá en la posición donde se soltó.

Seleccionar y mover

Selección simple

Haga clic sobre un elemento para seleccionarlo. Aparecerán los controles de redimensionado en los bordes y esquinas.

Selección múltiple

  • Shift + clic: agrega o quita elementos de la selección.
  • Marquee (recuadro): haga clic en un área vacía del canvas y arrastre para dibujar un recuadro de selección. Todos los elementos que intersecten serán seleccionados.

Mover elementos

Arrastre un elemento seleccionado para moverlo. Si hay varios seleccionados, todos se mueven juntos manteniendo su posición relativa.

Redimensionar

Arrastre los controles cuadrados en los bordes o esquinas del elemento seleccionado para cambiar su tamaño.

Bloquear / Desbloquear

Un elemento bloqueado (candado) no se puede mover ni redimensionar. Selecciónelo y desbloquéelo desde el panel de propiedades.

Tamaño de papel

En la barra superior del canvas, el selector Papel permite cambiar el formato de la hoja:

  • A4 (210 × 297 mm) — formato estándar de factura.
  • A5 (148 × 210 mm) — formato compacto.
  • Ticket 80 mm — impresora térmica estándar.
  • Ticket 58 mm — impresora térmica angosta.
  • Personalizado — ingrese ancho y alto en milímetros.

Al cambiar el tamaño, todos los elementos se redistribuyen y escalan proporcionalmente para mantener el diseño.

Zoom y navegación

Controles de zoom

  • Botones +/−: en la barra superior del canvas, aumentan o reducen el zoom en 10%.
  • Botón 1:1: restaura el zoom al 100%.
  • Ctrl + rueda del mouse: zoom suave centrado en la posición del cursor.
  • Pinch en trackpad: gesto de pellizco para acercar o alejar.

Rango

El zoom va desde 25% (vista general) hasta 200% (detalle fino).

Desplazamiento

Use la rueda del mouse (sin Ctrl) o las barras de scroll para desplazarse por el canvas cuando el contenido excede el área visible.

Panel de propiedades

Al seleccionar un elemento, el panel derecho muestra sus propiedades organizadas en pestañas:

General

  • Nombre: identificador del elemento.
  • Posición (X, Y): en milímetros desde la esquina superior izquierda de la sección.
  • Tamaño (Ancho, Alto): dimensiones en milímetros.
  • Bloqueado: impide mover o redimensionar.

Estilo

  • Tipografía (familia, tamaño, negrita, cursiva).
  • Alineación de texto (izquierda, centro, derecha).
  • Color de fondo y bordes.

Datos (solo Campo)

  • Campo vinculado: seleccione el dato del comprobante que mostrará este campo desde la lista categorizada.

Edición múltiple

Al seleccionar varios elementos, el panel muestra las propiedades comunes. Los campos con valores diferentes muestran "Mixto". Los cambios se aplican a todos los elementos seleccionados.

} @if (helpSection === 'data') {

Campos de datos

Los campos de tipo Campo se vinculan a datos del comprobante. Al seleccionar uno, en la pestaña "Datos" del panel de propiedades verá un listado categorizado:

Categorías disponibles

  • Empresa: nombre, RUT, dirección, teléfono, logo.
  • Comprobante: tipo, nombre, número, serie, fecha, forma de pago.
  • Cliente: nombre, documento, dirección, teléfono.
  • Moneda y Totales: símbolo de moneda, subtotal, impuestos, total.
  • Línea de Detalle: código, descripción, cantidad, precio, descuento, importe.
  • CFE (Comprobante Fiscal Electrónico): CAE, vencimiento, seguridad.
  • Contexto: fecha de impresión, número de página.

También puede escribir manualmente una ruta de binding personalizada.

} @if (helpSection === 'export') {

Exportación

JSON

El botón JSON descarga la plantilla completa en formato JSON. Este archivo contiene toda la configuración del diseño y puede usarse para:

  • Respaldo de la plantilla.
  • Importar en otro entorno o empresa.
  • Versionado manual del diseño.

Exportar HTML

El botón Exportar HTML genera un archivo HTML con directivas Zureo, listo para ser renderizado por el motor de impresión (Puppeteer) en el servidor.

Vista Previa

La pestaña Vista Previa muestra cómo se verá el comprobante con datos de ejemplo, permitiendo validar el diseño antes de exportar.
